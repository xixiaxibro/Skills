# Journal Best Practices

## Rewrite Rules

- Keep the user's original meaning; improve clarity, not stance.
- Preserve concrete facts (people, tools, outcomes, numbers).
- Place cause before emotion when possible.
- Avoid generic emotional statements without context.

## Quality Checklist

- Can a third party explain what happened from `今天发生了什么` alone?
- Does each feeling line map to a specific event or insight?
- Are learning points framed as reusable principles?
- Are next actions small, testable, and time-bounded?
- Does the entry still sound like the user's voice?

## Visual Presentation Rules

- Keep Markdown as canonical data source.
- Prefer offline rendering that works by double-clicking.
- Regenerate data files after edits before opening the board.
